Juti Noppornpitak
=================

My name is Juti Noppornpitak and I am a mathematician, software engineer, user
experience designer and photographer who currently live in Toronto yet frequently
travel to Thailand and Japan.

Currently, I work as a full-time senior PHP developer at `Instaclick Inc. <http://instaclick.com>`_
in Toronto, the main contributor of Imagination Framework (`shiroyuki/Imagination <https://github.com/shiroyuki/Imagination>`_),
Tori Framework (`shiroyuki/Tori <https://github.com/shiroyuki/Tori>`_) and a
time-to-time contributor to Symfony Framework (`symfony <https://github.com/symfony/>`_).
